# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices based on features such as **area, number of bedrooms, and bathrooms** using a **Linear Regression model**.

It demonstrates a complete machine learning workflow including data preprocessing, model training, evaluation, and visualization.

---

## 🚀 Features

* 📊 Exploratory Data Analysis (EDA)
* 🧹 Data cleaning and preprocessing
* 🤖 Linear Regression model training
* 📈 Model evaluation (MAE, R² Score)
* 📉 Residual analysis for error understanding
* 🧾 User input prediction system

---


## 📊 Model Performance

* **Mean Absolute Error (MAE):** *1.7346400287774695*
* **R² Score:** *0.9942779382174936*
